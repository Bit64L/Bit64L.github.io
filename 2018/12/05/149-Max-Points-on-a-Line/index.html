149. Max Points on a Line

发表于 2018-12-05 | | 阅读次数:

Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

Example 1:

Input: [[1,1],[2,2],[3,3]]
Output: 3
Explanation:
^
|
| o
| o
| o
+————->
0 1 2 3 4
Example 2:

Input: [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]
Output: 4
Explanation:
^
|
| o
| o o
| o
| o o
+——————->
0 1 2 3 4 5 6

Solution

本题解法并不困难，只需要通过计算斜率，统计斜率一样的点来获得在一条直线上最多的点数，但是毕竟是一道Hard题目，注意点还是挺多的。

注意点：

  1. 斜率
    斜率分为三种，水平为0，竖直为无穷大，其余通过dy-dx即可。
    由于精度问题，不采用浮点数保存斜率k，用最简分式表示斜率，且水平为(0,1),竖直为(1,0)
  2. 重复的点
    用字段dup来记录

    Complexity

    O(N2) O(N)
