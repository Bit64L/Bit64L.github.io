50. Pow(x, n)

发表于 2018-12-02 | | 阅读次数

Implement pow(x, n), which calculates x raised to the power n (xn).

Example 1:

Input: 2.00000, 10
Output: 1024.00000
Example 2:

Input: 2.10000, 3
Output: 9.26100
Example 3:

Input: 2.00000, -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
Note:

-100.0 < x < 100.0
n is a 32-bit signed integer, within the range [−231, 231 − 1]

Solution

解法1：递归法
当n为偶数时，res = myPow(n/2) myPow(n/2)，因为子问题重叠，可以用tmp记录下myPow(n/2)，然后res = tmp tmp;
当n为奇数时，res = x myPow(n/2) myPow(n/2),同样可用tmp来缓存myPow(n/2)的值。

解法2：迭代法
将指数n看做一个二进制数，二进制数为1是，需要将product乘入res。product维护x^(2i)。

细节：

  1. n由负数转化为整数时，可能会溢出，需要将n扩展为long型
  2. 指数为0时，一切皆为1

Complexity

O(logN) O(1)
O(logN) O(1)
