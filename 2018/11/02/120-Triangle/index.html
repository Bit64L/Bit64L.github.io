120.Triangle

发表于 2018-11-02 | | 阅读次数

本题是一道比较经典DP题目，曾记得在本科算法讲DP问题的时候就做过这道题，还有一次是在蓝桥杯国赛上遇到了这道题，可惜当时水平有限，直接就跳过了没敢做，如果做的话可能国二就变成国一了。
本文将给出Top Down和Bottom Up两种解法。

Given a triangle, find the minimum path sum from top to bottom. Each step you may move to adjacent numbers on the row below.

For example, given the following triangle

[
[2],
[3,4],
[6,5,7],
[4,1,8,3]
]
The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

Note:

Bonus point if you are able to do this using only O(n) extra space, where n is the total number of rows in the triangle.

Bottom Up

dp[i][j]代表从底部走到nums[i][j]这个元素的最小和，dp式为：
dp[i][j] = min(dp[i+1][j+1],dp[i+1][j]) + nums[i][j]

进一步分析，dp[i]和dp[i-1]有关系，因此可以降维至一维数组，使dp只保存下一行的最小值，dp式:
dp[j] = min(dp[j+1],dp[j]) + nums[i][j];

O(N2) O(N)

Top Down

分析过程类似Bottom Up，有一点不同的是，为防止dp数组被错误覆盖，需要从后往前更新，dp式：
dp[j] = min(dp[j],dp[j-1]) + nums[i][j]

O(N2) O(N)
