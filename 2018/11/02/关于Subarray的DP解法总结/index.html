关于Subarray的DP解法总结

发表于 2018-11-02 | | 阅读次数

做了两道关于Subarray的题目，发现是有些套路的，于是写下此篇总结。
Subarray是连续的，而Subsequence是不连续的。

53 Maximum Subarray

Given an integer array nums, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.

Example:

Input: [-2,1,-3,4,-1,2,1,-5,4],
Output: 6
Explanation: [4,-1,2,1] has the largest sum = 6.
Follow up:

If you have figured out the O(n) solution, try coding another solution using the divide and conquer approach, which is more subtle.

可以使dp[i]代表以nums[i]结尾的subarray的最大和，则有dp式：
dp[i] = max(nums[i], nums[i]+dp[i-1])。
进一步优化：可以发现dp[i]只可能用到dp[i-1]的值，则可以维护一个max变量代表dp[i-1]，可以使空间复杂度降到O(N)，则新的dp式为:
max = max(nums[i], nums[i]+max)

O(N) O(1)

152 Maximum Product Subarray

Given an integer array nums, find the contiguous subarray within an array (containing at least one number) which has the largest product.

Example 1:

Input: [2,3,-2,4]
Output: 6
Explanation: [2,3] has the largest product 6.
Example 2:

Input: [-2,0,-1]
Output: 0
Explanation: The result cannot be 2, because [-2,-1] is not a subarray.

此题相较于53题有些许变化。分析过程如下：
使dp[i]表示以nums[i]结尾的subarray的最大值，但是当nums[i]小于0时，dp[i-1]nums[i]为以nums[i]结尾的长度大于1的subarray的最小值，不符合我们要的以nums[i]结尾的长度大于1的subarray的最大值。若要满足我们的需求，则需要nums[i]<0时，nums[i] 以nums[i-1]为结尾的subarray的最小值，nums[i]>=0时，nums[i] *以nums[i-1]为结尾的subarray的最大值。

由此可知，dp[i]要保存min和max两个值，因为dp[i]只可能用到dp[i-1]，则设置两个变量min和max即可，这样使得空间复杂度降到O(1)

O(N) O(1)
