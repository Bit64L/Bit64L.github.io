23.Merge k Sorted Lists

发表于 2018-11-08 | | 阅读次数

Merge Two Sorted Lists的高级版

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

Example:

Input:
[
1->4->5,
1->3->4,
2->6
]
Output: 1->1->2->3->4->4->5->6

解法一：类似Merge 2 Sorted Lists， 维护K个指针，每次在K个指针中取最小的结点加入目标链表末尾，直至K个指针均为NULL.

解法二：进一步优化：每次从K个结点中取最小的一个，可以想到可以用最小堆来解决。
细节：由于ListNode不是Java内置类，使用优先级队列时需要传入一个比较器Comparator，可以用匿名类实现

解法三：分支法（待实现）

  1. O(KN) O(k)
  1. 堆 O(Nlogk) O(k)
