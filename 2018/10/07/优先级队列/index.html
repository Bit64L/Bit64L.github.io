优先级队列

发表于 2018-10-07 | | 阅读次数:

许多时候，我们需要处理有序的元素，但不需要他们完全有序，或者说不一定要一次性就将它们完全排序，这时候就可以使用优先级队列解决。

定义

优先级队列支持两种基本操作：

  • 删除最大(最小)元素
  • 插入新元素

Plblic class MaxPQ<Key extends Comparable<Key>>

类型 方法 说明
MaxPQ 创建一个优先队列
MaxPQ(int max) 创建一个初始容量为max的优先队列
MaxPQ(Key[] a) 用a[]中的元素创建一个优先队列
void insert(Key v) 插入新元素
Key dekMax 删除并返回最大元素
boolean isEmpty() 返回队列是否为空
int size() 返回队列大小

实现方式

数据结构 插入元素 删除元素
有序数组 N 1
无序数组 1 N
堆 logN logN

有序数组(链表)实现

insert()时采用插入排序的方法
可以先使用binary search找到插入点
则有N次access和logN次compares

无序数组(链表)实现

删除最大元素时需要添加一段类似选择排序的代码

二叉堆实现

  • 二叉堆分最大堆(max-heap)和最小堆(min-heap)
  • 满二叉树：一棵深度为k，且有2^k-1个节点的树是满二叉树。
  • 完全二叉树(full binary tree)：完全二叉树是由满二叉树而引出来的。对于深度为K的，有n个结点的二叉树，当且仅当其每一个结点都与深度为K的满二叉树中编号从1至n的结点一一对应
